# VERP - Venue Event Resource Planning

A comprehensive ERP system for venue management with 8-role RBAC (Role-Based Access Control) system.

## Features

- **8-Role System**: Super Admin, Branch Manager, Sales/Front Desk, Accountant, Operations Manager, Vendor, Security Staff, Customer
- **Customer Portal**: Self-service booking, invoices, and profile management
- **Internal Portal**: Complete venue management with role-based permissions
- **RBAC Security**: Page-level access control with granular permissions (VCEDA$X)

## Quick Start

1. Clone the repository
2. Open `index.html` in a web browser for the customer portal
3. Navigate to `/internal/` for the internal staff portal

## Login Credentials (Demo)

Access the internal portal at `/internal/index.html` with any of these roles:
- Super Admin (sa)
- Branch Manager (bm)
- Sales / Front Desk (sl)
- Accountant (ac)
- Operations Manager (op)
- Vendor (vn)
- Security Staff (se)
